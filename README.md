# 🌐 Kişisel Web Projem

Bu proje, Web Teknolojileri dersi kapsamında geliştirilmiştir.  
Amaç, HTML, CSS, JavaScript ve PHP kullanarak kişisel bir web sitesi oluşturmaktır.

## 📄 Sayfalar

- **Hakkımda (index.html):** Kişisel bilgiler ve tanıtım
- **CV (cv.html):** Eğitim ve yetenekler (semantic HTML kullanılmıştır)
- **Şehrim:** Tanıtım ve görseller
- **Miras / Takım:** Kültürel veya sportif içerik
- **İlgi Alanlarım:** API kullanılarak veri çekme
- **İletişim:** Form ve doğrulama işlemleri
- **Login:** PHP ile giriş sistemi

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- Bootstrap
- PHP

## 🚀 Proje Amacı

- Web geliştirme becerilerini göstermek  
- Responsive ve kullanıcı dostu bir site oluşturmak  
- Frontend ve backend teknolojilerini birlikte kullanmak
